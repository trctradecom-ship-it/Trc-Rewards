What is TRC EMA Price?

TRC EMA (Exponential Moving Average) Price is a smart price smoothing system that reduces sudden market spikes and dumps.

Formula:
EMAnew = ((Raw Price × 20) + (Old EMA × 80)) / 100

Example 1:
Old EMA Price = $10
New Market Price = $15

EMAnew = ((15 × 20) + (10 × 80)) / 100
EMAnew = (300 + 800) / 100
EMAnew = 1100 / 100
EMA New price = $11

Example 2:
Old EMA Price = $11
New Market Price = $20

EMAnew = ((20 × 20) + (11 × 80)) / 100
EMAnew = (400 + 880) / 100
EMAnew = 1280 / 100
EMA New price = $12.8

Example 3 (Price Drop):
Old EMA Price = $12.8
New Market Price = $15

EMAnew = ((15 × 20) + (12.8 × 80)) / 100
EMAnew = (300 + 1024) / 100
EMAnew = 1324 / 100
EMA New Price = $13.24

Example 4 (Price Drop):
Old EMA Price = $13.24
New Market Price = $10

EMAnew = ((10 × 20) + (13.24 × 80)) / 100
EMAnew = (200 + 1059.2) / 100
EMAnew = 1259.2 / 100
EMA New Price = $12.592

EMA moves gradually instead of instantly following sudden market changes, helping create a more stable and fair TRC price.

Benifits Of TRC Affiliate Staking Program use Trc EMA Price for level Joining Instead Of DEX Trc Price?

In the TRC system, users join levels using the TRC equivalent value of USD. This means the required TRC amount changes based on the current TRC market price.

If direct raw DEX price is used, sudden price pumps or dumps can create imbalance in level costs.

Example:
Level Cost = $100

TRC Price = $10 → User pays 10 TRC
TRC Price suddenly drops to $5 → User pays 20 TRC
TRC Price suddenly pumps to $20 → User pays only 5 TRC

This creates unstable joining conditions because users entering at different times may pay very different TRC amounts for the same level.

TRC EMA Price helps smooth these sudden market changes by gradually adjusting the joining price instead of instantly following every DEX fluctuation.

Benefits of EMA system:
• Creates more stable level costs
• Reduces sudden TRC amount imbalance
• Protects against extreme DEX volatility
• Supports fairer user entry conditions
• Helps maintain long-term ecosystem stability
